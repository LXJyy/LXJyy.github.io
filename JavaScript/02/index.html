“只有经过九世的孤独，你才能成为真正的强者。”虚空建言。

“我宁愿这十世换取她一世的幸福。”身着黑色铠甲的少年坚毅地说道。“你要知道你就算换来了她的一世，她也会因各种原因而不幸福。”虚空说。“那我便守在她的身边永生永世，直到她找到自己的幸福。”少年毫不犹豫地说。虚空沉默半晌，问少年：“值得吗？彦延。”“值，为她，付出一切都值。”“好吧，我成全你。千年之后她会在云市出现。那时的她早就忘了你了吧。彦延，一路走好。”虚空说。彦延挥挥手，消失在无尽黑暗中。

千年后，云市。

彦延化作一普通少年来到此地，他的眼中带着一丝温柔，笑着喃喃：“小娜，我来找你了。”一步一步在人群中消失了踪影。

一位衣着朴素的少女苦恼地看着令人头疼的功课。嘴里嘟囔着：“这么难，有那么多。怎么写得完？”少女不由泄气，随后又燃起斗志。紧握拳头，说“没有什么是可以难得到我林娜的，加油。”为自己鼓了鼔气，拿起手中的笔，沙沙地写了起来。一张张演算纸不断被写满，堆积在林娜的脚边。当脚边的纸堆积得像小山时，林娜捧着这些东西到回收站那，将这些投入进去，等待着新的演算纸生成。林娜也没闲着，在心里不断演算的刚刚未演算完的式子。沉浸在演算中的她在他的眼里是最美的。娜，我们很快就会见面了。彦延深深地望了林娜一眼，往院长办公室走去。轻轻地叩了叩门，开门走进了院长办公室。映入眼中的是一位正喝着真灵茶的戴着金丝边眼镜的成熟男子，成熟男子微笑着朝彦延打了声招呼：“呦，彦延。千年未见了吧。”彦延错愕了，他很吃惊：“虚空，你怎么来了？”虚空笑了笑，说：“你也知道我就一个人呆着那个地方也无聊。所以特意找了份工作，现在我是这个学院的院长，请叫我李凌虚院长，彦延童鞋。”虽然有点难以置信，但彦延不得不接受这个事实。李凌虚，也就是虚空，正色说：“现在我是这个学院的院长，所以为了保护这个学院的花花草草，我要与你约法三章。第一，不得使用能力。第二，不得告诉你口中的那个她，你的身份以及她的事情。第三，好好地照顾好自己，毕竟你是我唯一的朋友。所以综上所述，请戴上这个手表。这谁我为你特意准备的禁能手表。同意你就可以入学了，并且去她所在的那个班哦。”果断的，彦延把手表戴到了手上，一条条法则交织起来，彦延的能力暂时给封印住了。虚空正色对彦延说“彦延，你要知道你所付出的代价太大了。你只有一个月的时间。超过一个月，你就会魂飞魄散。这世上就不会再有你的存在，你要切记，这世上我只有你一个朋友。”彦延默然点了点头，说：“虚空，你知道的，为了她，我会不顾一切。只愿她能够一切安好。”“唉，你去吧。我也就不管你了，真是烦死了。”虚空摆了摆手。

“走了，虚空如果我出了什么事。帮我照顾好她，拜托了。”彦延说着带上了门。虚空落寞地说：“彦延，你的事情命中注定。如果你不成功，今后我又要一个人了。是福不是祸，是祸躲不过。”命运的齿轮开始运转了。
